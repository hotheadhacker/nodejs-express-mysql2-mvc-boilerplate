## Deployment

To deploy this project run

```git
  git clone https://github.com/hotheadhacker/nodejs-express-mysql2-mvc-boilerplate.git
```
```bash
  cd nodejs-express-mysql2-mvc-boilerplate
```
```bash
  npm install
```
```bash
  npm start
```
